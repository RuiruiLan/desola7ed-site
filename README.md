# DESOLA7ED — Artist Website

A single static page (`index.html`) used as the official artist website for Spotify for Artists verification. No backend, no build step.

## Deploy with Netlify Drop (fastest, ~1 minute)

1. Go to https://app.netlify.com/drop
2. Drag the site folder (just `index.html`) onto the page.
3. Netlify gives you a live URL immediately (e.g. `https://random-name.netlify.app`).
4. Optional: sign in and rename the site to something like `desola7ed.netlify.app` under **Site settings → Change site name**.
5. Submit that URL to Spotify for Artists.

## Deploy with GitHub Pages (free, permanent)

1. Create a new public repository on GitHub (e.g. `desola7ed-site`).
2. Upload `index.html` to the repository root (drag-and-drop works on github.com via **Add file → Upload files**).
3. Go to **Settings → Pages**.
4. Under **Source**, choose **Deploy from a branch**, select `main` and `/ (root)`, then save.
5. After a minute or two the site is live at:
   `https://<your-username>.github.io/desola7ed-site/`
6. Submit that URL to Spotify for Artists.

## Updating the page

Edit `index.html` and re-upload (Netlify Drop: drag the folder again; GitHub Pages: commit the new file). Changes go live automatically.
